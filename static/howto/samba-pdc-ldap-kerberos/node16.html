Configurazione NTP

Configuriamo NTP modificando il file /etc/ntp.conf: 
server   127.127.1.0
fudge 127.127.1.0 stratum 10
server 192.168.0.1
driftfile /etc/ntp/drift
multicastclient
broadcastdelay 0.008
e avviamolo 
# /etc/init.d/ntp start
