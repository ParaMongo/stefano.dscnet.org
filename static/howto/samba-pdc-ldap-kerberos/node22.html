Join al reame Kerberos

Possiamo ora aggiungere il client al reame Kerberos: 
# kinit kadmin/admin
# kadmin add --random-key --max-ticket-life=unlimited --max-renewable-life=unlimited --expiration-time=never \
   --pw-expiration-time=never --attributes= host/c-lnx-001.esempio.lan
# kadmin add --random-key --max-ticket-life=unlimited --max-renewable-life=unlimited --expiration-time=never \
   --pw-expiration-time=never --attributes= host/c-lnx-001
# kadmin add --random-key --max-ticket-life=unlimited --max-renewable-life=unlimited --expiration-time=never \
   --pw-expiration-time=never --attributes= cifs/c-lnx-001.esempio.lan
# kadmin add --random-key --max-ticket-life=unlimited --max-renewable-life=unlimited --expiration-time=never \
   --pw-expiration-time=never --attributes= cifs/c-lnx-001
# ktutil -k /etc/krb5.keytab get -p kadmin/admin@ESEMPIO.LAN host/c-lnx-001.esempio.lan
# ktutil -k /etc/krb5.keytab get -p kadmin/admin@ESEMPIO.LAN host/c-lnx-001
# ktutil -k /etc/krb5.keytab get -p kadmin/admin@ESEMPIO.LAN cifs/c-lnx-001.esempio.lan
# ktutil -k /etc/krb5.keytab get -p kadmin/admin@ESEMPIO.LAN cifs/c-lnx-001
